h1. Webster

Generate random short words. Good for human-readable confirmation codes.

h2. Install

  sudo gem install dancroak-webster --source=http://gems.github.com

h2. Usage

  require 'rubygems'
  require 'webster'

  include Webster	
  random
  
  => "spoonless" 

h2. Shoulda macro

   context "Subscription#activation_code" do
    setup do
      subscription = Factory :subscription
      @word = subscription.set_activation_code
    end
  
   should_be_webster_word "@word"
